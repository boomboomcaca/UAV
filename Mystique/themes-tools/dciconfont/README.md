## 德辰 iconfont 图标库组件

    使用方法和 antd icon 类似，修改字体大小需要通过iconSize属性修改， 颜色通过设置 color属性 修改

    可以传入onClick事件

```javascript
<RefreshIcon iconSize={36} color="#fff" onClick={() => {}} />
```

按钮

## 何时使用

- 在使用点击触发某种事件时使用

## API

## Button props

按钮的属性说明如下：

属性 | 说明 | 类型 | 默认值
disabled | 按钮失效状态 | boolean | false
size | 设置按钮大小 | large | middle | small | middle
type | 设置按钮类型 | primary | default | default
onClick | 点击按钮时的回调 | (event) => void | -

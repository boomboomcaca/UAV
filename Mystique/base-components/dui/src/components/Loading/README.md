- 例子详见 Demo

## API

### Loading props

| 参数        | 说明             | 类型   | 默认值                            |
| ----------- | ---------------- | ------ | --------------------------------- |
| className   | 顶层 div 样式    | css    | null                              |
| loadingSize | 加载图标大小     | any    | 24                                |
| loadingMsg  | 加载信息         | string | ''                                |
| loading     | 加载中           | bool   | true                              |
| vertical    | 图标文字垂直顺序 | bool   | false                             |
| type        | 图标类型         | string | colorful,single,gradient(default) |
